# Minimal, legible plot theme (uses family tones for accents)

Minimal, legible plot theme (uses family tones for accents)

## Usage

``` r
theme_albers(family = "red", base_size = 13, base_family = "system-ui")
```

## Arguments

- family:

  Palette family used by companion scales.

- base_size:

  Base font size.

- base_family:

  Base font family.
