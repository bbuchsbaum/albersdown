# Scales that use the family's tones (discrete/continuous)

Scales that use the family's tones (discrete/continuous)

## Usage

``` r
scale_color_albers(family = "red", discrete = TRUE, ...)
```

## Arguments

- family:

  Palette family.

- discrete:

  Whether to use a discrete palette; if FALSE, uses a gradient.

- ...:

  Passed to underlying `ggplot2` scale.
